# passible

**Passive L4–L7 Network Profiler** для обнаружения бекдоров, майнеров и подозрительной сетевой активности.

`passible` — это легковесный демон на C, который пассивно наблюдает за исходящими сетевыми соединениями на хосте с помощью eBPF, применяет правила из конфигурации и генерирует алерты при обнаружении аномалий: подозрительных портов, регулярных «heartbeat»-соединений, активности недоверенных процессов и других признаков вредоносного поведения.

Работает без перехвата payload по умолчанию, минимизируя overhead. Конфигурируется через `/etc/passible/conf.yml`. Результаты записываются в JSONL-лог или экспортируются в Prometheus.

Предназначен для Linux-хостов с ядром ≥5.8 и поддержкой BTF.
